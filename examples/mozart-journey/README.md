# Mozart Journey Example

This case study is migrated from the Mozart Journey static page in the Moltpany portfolio. It demonstrates Agent-Mappy's first target shape:

- one JSON array of map story entries;
- city and precise-place coordinates;
- year-based timeline data;
- source labels, URLs, and summaries;
- optional images and listening search links;
- personal collection ids for favorites and playlists.

Validate it from the repository root:

```bash
node tests/validate.js
```

The original page is a Chinese static webpage about Mozart's travels, cities, years, works, and concise historical meaning. Agent-Mappy treats it as the first reusable example, not as the only supported topic.
